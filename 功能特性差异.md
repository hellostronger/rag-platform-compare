## 3. 各平台核心特色

### Dify
- **完整的工作流编排能力**: 可视化编排，支持多种节点类型
- **丰富的索引处理器**: 支持段落索引、父子索引、问答索引等
- **完善的 RAG 管道**: 从文档处理到检索的完整流程
- **多数据集路由**: 支持 React 路由和函数调用路由

### RAGFlow
- **深度文档理解**: DeepDoc 模块提供强大的文档解析能力
- **GraphRAG**: 内置知识图谱构建和查询能力
- **高级检索增强**: TOC 增强、多语言交叉检索
- **灵活的 Agent 组件**: 丰富的可组合组件库

### Coze Studio
- **企业级架构**: DDD 设计的微服务架构
- **一站式平台**: 从开发到部署的完整工具链
- **丰富的插件生态**: 支持多种第三方服务集成
- **高性能检索**: 基于 Elasticsearch 和 Milvus

### QAnything
- **父子文档检索**: ParentDocumentRetriever 实现
- **混合检索优化**: Milvus + Elasticsearch 双路召回
- **简洁的架构**: 专注于检索问答核心功能
- **一键部署**: Docker Compose 快速部署

### WeKnora
- **企业级解决方案**: 面向企业的完整 RAG 平台
- **文档处理链**: 支持多种文档格式和解析引擎
- **分块器支持**: 可配置的分块策略
- **多语言支持**: 国际化支持良好

### FastGPT
- **可视化 AI 应用构建**: 直观的工作流编辑
- **丰富的插件系统**: 支持自定义工具和服务
- **完整的文档体系**: 详细的使用和开发文档
- **多模型支持**: 支持主流大模型提供商

## 4. 总结

各平台在功能上各有侧重:
- **Dify** 和 **RAGFlow** 在 RAG 核心能力上最为完善
- **Coze Studio** 提供了企业级的完整解决方案
- **QAnything** 专注于检索问答的核心体验优化
- **WeKnora** 和 **FastGPT** 提供了良好的用户友好的开发体验
